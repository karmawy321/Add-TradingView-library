The Execution Model

FractalScript follows the Bar-by-Bar execution model. Every time the chart updates, the engine runs your entire script for every single candle on the chart, from left to right. This allows you to "remember" data from the past using history references or persistent variables.

Every script in this guide is Pine Script v5 compatible — paste any example directly into TradingView and it will produce identical results. Use str.tostring(value, "#.##") for all number-to-text formatting to stay compatible on both platforms.

The Toolbelt

Technical Analysis

Production-grade math functions for market analysis.

  • ta.rsi / ta.stoch (Oscillators)

Core Logic

The building blocks of your strategy logic.

  • var (Persistent variable across bars)
  • [n] (History reference, e.g. close[1])
  • ta.crossover / ta.crossunder (Signal logic)
  • array.* (Dynamic data storage)

Visual Control

Direct the canvas to draw your insights.

  • line.new (X/Y coordinates, custom styles)
  • plot / plotshape (Series & marker plotting)
  • color.* (Lime, Red, Aqua, Gold, Silver)
  • extend.* (Right, Left, Both, None)
  • line.style_* (Solid, Dashed, Dotted)
  • overlay=false (Separate sub-pane rendering)

Label System

Place and mutate text annotations at any price level.

  • label.new(x, y, text) (Create annotation)
  • label.set_text / label.set_xy (Mutate in-place)
  • label.set_color / label.set_textcolor (Style)
  • label.set_size (tiny, small, normal, large, huge)
  • label.delete(id) (Remove a label)
  • label.style_label_up/down/left/right (Arrow styles)
  • label.style_circle/square/diamond/flag (Markers)

String Tools

Format numbers and build text for labels and outputs.

  • str.tostring(val) (Number to string)
  • str.tostring(val, "#.##") (Formatted decimal)
  • str.tostring(val, "#.0000") (Fixed precision)
  • str.length / str.upper / str.lower (Text ops)
  • "Label: " + str.tostring(n) (Concatenation)

Labels & Annotations

label.new places a text annotation at any bar and price. Unlike plotshape, labels carry arbitrary text — combine them with str.tostring() to display live values. Use var label + label.set_* to mutate a single label across bars instead of creating thousands of new ones.

The max_labels_count parameter in indicator() controls how many labels persist on screen at once (default 50, max 500). Older labels are automatically garbage-collected when the limit is reached.

1. The Golden Cross Sniper
A classic trend-following strategy using two moving averages. Draws a signal when the fast trend crosses the slow trend.
//@version=5 indicator("Golden Cross", overlay=true) fast = ta.ema(close, 50) slow = ta.ema(close, 200) plot(fast, color=color.aqua, linewidth=2) plot(slow, color=color.orange, linewidth=2) long = ta.crossover(fast, slow) plotshape(long, style=shape.triangleup, location=location.belowbar, color=color.green, text="BUY")
2. RSI Overbought / Oversold
Identifies momentum extremes. Draws a horizontal support level when RSI is oversold and resistance when overbought.
//@version=5 indicator("RSI Extremes", overlay=true) rsiVal = ta.rsi(close, 14) if rsiVal > 70 line.new(bar_index - 5, high, bar_index, high, color=color.red, width=2, extend=extend.right) if rsiVal < 30 line.new(bar_index - 5, low, bar_index, low, color=color.green, width=2, extend=extend.right)
3. ATR Volatility Bands
Draws dynamic channels based on market volatility. Great for setting stop losses or identifying break-outs.
//@version=5 indicator("ATR Channels", overlay=true) ma = ta.sma(close, 20) atr = ta.atr(14) upper = ma + (atr * 2) lower = ma - (atr * 2) line.new(bar_index-1, upper[1], bar_index, upper, color=color.aqua) line.new(bar_index-1, lower[1], bar_index, lower, color=color.orange)
4. VWAP Trend Follower
Uses Volume Weighted Average Price (VWAP) to find the institutional trend. Accurate only on charts with volume data.
//@version=5 indicator("VWAP Power", overlay=true) v = ta.vwap(close, volume) isBull = close > v line.new(bar_index-1, v[1], bar_index, v, color=isBull ? color.lime : color.red, width=2)
5. MACD Momentum Histogram
Calculates the full MACD suite. Draws vertical "bars" using lines to represent momentum strength.
//@version=5 indicator("MACD Visual", overlay=true) // Calculate MACD manually for parser safety macdLine = ta.ema(close, 12) - ta.ema(close, 26) signalLine = ta.ema(macdLine, 9) hist = macdLine - signalLine // Draw histogram as vertical lines from high/low if bar_index % 2 == 0 col = hist > 0 ? color.lime : color.red line.new(bar_index, high, bar_index, high + (hist * 10), color=col, width=2)
6. Infinite Support Grid
Tests the infinite clipping logic. Automatically draws a horizontal grid at every 100-point interval.
//@version=5 indicator("Level Grid", overlay=true) // Using simple price to demonstrate infinite extension curPrice = close prevPrice = close[1] if curPrice != prevPrice line.new(bar_index, curPrice, bar_index+1, curPrice, color=color.white, extend=extend.both)
7. Swing High/Low Breakouts
Identifies structural break-outs. Plots a marker when price closes above the highest point of the last 50 bars.
//@version=5 indicator("Structure Break", overlay=true) high50 = ta.highest(high, 50) if ta.crossover(close, high50[1]) plotshape(true, style=shape.labelup, location=location.belowbar, color=color.aqua, text="BOS")
8. Rainbow Cascade
Stress test for the renderer. Draws multiple offset moving averages to create a "ribbon" effect.
//@version=5 indicator("MA Ribbon", overlay=true) for i = 1 to 10 ma = ta.sma(close, i * 10) line.new(bar_index-1, ma[1], bar_index, ma, color=color.new(color.aqua, i * 5))

11. Swing Pivot Labels
Detects local swing highs and lows using neighbor comparison, then places a price label at each pivot. Demonstrates label.new + str.tostring for live price display. Works identically on TradingView Pine v5.
//@version=5 indicator("Swing Labels", overlay=true, max_labels_count=200) // Bar[2] is a pivot high if it's above both its neighbors isPivotHigh = high[2] > high[3] and high[2] > high[1] isPivotLow = low[2] < low[3] and low[2] < low[1] if isPivotHigh label.new(bar_index - 2, high[2], str.tostring(high[2], "#.##"), style=label.style_label_down, color=color.red, textcolor=color.white, size=size.small) if isPivotLow label.new(bar_index - 2, low[2], str.tostring(low[2], "#.##"), style=label.style_label_up, color=color.green, textcolor=color.white, size=size.small)
12. RSI Sub-Pane (overlay=false)
Renders RSI in its own dedicated pane below the price chart using overlay=false. The pane has its own scale (0–100). The color shifts red when overbought and green when oversold. Also draws the 70 and 30 reference lines.
//@version=5 indicator("RSI Panel", overlay=false) rsiVal = ta.rsi(close, 14) // Color shifts based on RSI zone col = rsiVal > 70 ? color.red : rsiVal < 30 ? color.lime : color.aqua plot(rsiVal, color=col, linewidth=2) // Overbought / oversold reference levels plot(70, color=color.new(color.red, 60)) plot(50, color=color.new(color.gray, 70)) plot(30, color=color.new(color.lime, 60))
13. MACD Signals via Tuple Destructuring
Uses tuple destructuring [a, b, c] = ta.macd() to unpack all three MACD values in one line — the same syntax supported by TradingView Pine v5. Colors the price line by MACD momentum direction and fires signal markers on crossovers.
//@version=5 indicator("MACD Signals", overlay=true) // Unpack all three MACD values in one line [macdLine, signalLine, hist] = ta.macd(close, 12, 26, 9) bullCross = ta.crossover(macdLine, signalLine) bearCross = ta.crossunder(macdLine, signalLine) // Price bar color follows MACD momentum plot(close, color=macdLine > signalLine ? color.lime : color.red, linewidth=2) plotshape(bullCross, style=shape.triangleup, location=location.belowbar, color=color.lime, size=size.small, text="M+") plotshape(bearCross, style=shape.triangledown, location=location.abovebar, color=color.red, size=size.small, text="M-")
14. Trend State Machine with Transition Labels
Demonstrates var persistence and := reassignment. A single integer tracks the current trend state across all bars. When the state flips, a label.new marks the exact bar of the transition — so you can see precisely when momentum changed.
//@version=5 indicator("Trend Engine", overlay=true, max_labels_count=100) fastMA = ta.ema(close, 9) slowMA = ta.ema(close, 21) // var persists the trend state across every bar var int trend = 0 if ta.crossover(fastMA, slowMA) trend := 1 if ta.crossunder(fastMA, slowMA) trend := -1 col = trend == 1 ? color.lime : trend == -1 ? color.red : color.gray plot(fastMA, color=col, linewidth=2) plot(slowMA, color=color.gray, linewidth=1) // Drop a label only on the bar where the trend flips if ta.crossover(fastMA, slowMA) label.new(bar_index, low, "BULL", style=label.style_label_up, color=color.lime, textcolor=color.black, size=size.normal) if ta.crossunder(fastMA, slowMA) label.new(bar_index, high, "BEAR", style=label.style_label_down, color=color.red, textcolor=color.white, size=size.normal)
